02 - Operators, Conditionals, Loops

0.1 + 0.2 = 0.30000000004

Lernziele dieser Lektion

  • Kennen der Operatoren und diese korrekt verwenden können.
  • Verstehen wie Loops funktionieren und diese selber schreiben können.
  • if/else-Conditionals korrekt anwenden können.

Operatoren

  • The Assignment Operator = assigns values
  • The Addition Operator + adds values
  • The Multiplication Operator * multiplies values
  • The Comparison Operator > compares values

Conditionals

  • Use if to specify a code block to be executed, if a specified condition is true
  • Use else to specify a code block to be executed, if the same condition is false
  • Use else if to specify a new condition to test, if the first condition is false
  • Use switch to specify many alternative code blocks to be executed
  • Use (? :) (ternary) as a shorthand for if...else

Arrays

Key characteristics of JavaScript arrays are:

  • Elements: An array is a list of values, known as elements.
  • Ordered: Array elements are ordered based on their index.
  • Zero indexed: The first element is at index 0, the second at index 1, and so on.
  • Dynamic size: Arrays can grow or shrink as elements are added or removed.
  • Heterogeneous: Arrays can store elements of different data types (numbers, strings, objects and other arrays).

Implementation

JavaScript wird in der Regel mit einer (oder mehreren) der folgenden Möglichkeiten in eine Webseite eingebunden.

  • Inline
  • Direkt im <head>
  • Als Skript in einer externen Datei

Inline

  • JavaScript kann inline direkt in einem HTML-Tag erfasst werden.
  • Einfach zu nutzen aber stark limitiert.
  • Kann nur bei bestimmten Events genutzt werden.
                    
                        <button onclick='alert("I got clicked!")'>Click me!</button>

Direkt im Head

  • Im <head> kann mithilfe eines Skript-Tags direkt JavaScript erfasst werden
  • Bietet (beinahe) das gesamte Arsenal von JavaScript
  • Für grosse Projekte schnell unübersichtlich
                    
                        <head>
                        <script>
                        const elements = document.querySelectorAll(".tile");
                        elements.forEach(e => {
                        e.addEventListener("click", event => {
                        e.classList.toggle("blink");
                        })
                        })
                        </script>
                        </head>

Externe Datei

  • Etwas aufändiger einzurichten
  • Bietet alle Features von JavaScript
  • Auch für grosse Projekte (oder die die es noch werden wollen) geeignet
  • Erlaubt das Im- und Exportieren von Code
                    
                        <script src="/path/to/my/file.js"></script>

TL;DR

  • JavaScript, die Programmiersprache des Webs, berechnet, manipuliert und validiert Daten.
  • Wurde innert 10 Tagen von Brendan Eich in 1995 entwickelt.
  • Integrationstypen:
    • Inline: Einfach, aber limitiert.
    • Direkt im Head: Alle JavaScript-Funktionalitäten, nicht gut skalierbar.
    • Externe Datei: Modular und skalierbar, ideal für grössere Projekte.
