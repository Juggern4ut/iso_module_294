01 - Einführung in JavaScript

Die Grundlagen

Lernziele dieser Lektion

  • Verstehen, was JS ist und welche Rolles es in der Webentwicklung spielt.
  • Verstehen der Grundlagen wie JavaScript mit einer Webseite interagiert.

Was ist JavaScript?

  • JavaScript ist die Programmiersprache des Webs.
  • Es berechnet, manipuliert und validiert Daten.
  • Es kann sowohl das HTML als auch das CSS der Webseite manipulieren.

Geschichte von JavaScript

  • Im Jahre 1995 von Brendan Eich in nur 10 Tagen entwickelt.
  • 99% aller Webseiten nutzen JavaScript auf Client-Seite.
  • Dank Node.js auch immer öfters Serverseitig genutzt.
  • Alle modernen Web-Frameworks (Angular, Nuxt, React...) bauen auf JavaScript auf.

Implementation

JavaScript wird in der Regel mit einer (oder mehreren) der folgenden Möglichkeiten in eine Webseite eingebunden.

  • Inline
  • Direkt im <head>
  • Als Skript in einer externen Datei

Inline

  • JavaScript kann inline direkt in einem HTML-Tag erfasst werden.
  • Einfach zu nutzen aber stark limitiert.
  • Kann nur bei bestimmten Events genutzt werden.
                    
                        <button onclick='alert("I got clicked!")'>Click me!</button>

Die meistgenutzen Tags

Tagname Verwendungszweck
h1, h2, h3 ... Überschriften in unterschiedlichen grössen
p Einfacher Paragraf, um Textinhalte darzustellen
a Link (Anchor), um zu anderen Seiten oder Inhalten zu verlinken.
img Bild, um visuelle Inhalte einzufügen.
ol, ul (Nummerierte) Listen, um Inhalte zu strukturieren.
li Listenelement in (nummerierten) Listen.
div Allgemeiner Container um Inhalte zu gruppieren.
main, section, article, aside Bestimmte Container

Head- und Body-Tag

Jedes HTML-Dokument besteht aus zwei Hauptteilen: Dem <head> und dem <body>.

  • Der <head> enthält Metadaten über das Dokument, wie den Titel, Skripte und andere Informationen. Diese Elemente sind im Browser nicht sichtbar.
  • Der <body> enthält den eigentlichen Inhalt der Webseite, der im Browser dargestellt wird.

Struktur einer Webseite

Eine grundlegende Webseite sollte immer mindestens die folgende Struktur aufweisen 

                    
                        <!DOCTYPE html> 
                        <html> 
                        <head> 
                        ...
                        </head>
                        <body> 
                        ...
                        </body>
                        </html>
