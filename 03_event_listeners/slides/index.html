03 – Funktionen & Events

Funktionen, Arrow-Functions und EventListeners

Lernziele dieser Lektion

  • Verstehen, was Funktionen sind und wieso sie genutzt werden
  • Den Aufbau einer Funktion kennen
  • Return-Werte verstehen und anwenden
  • Die gängigsten Funktionstypen kennen
  • Wissen, was EventListener sind
  • Gängige Events kennen und anwenden
  • Events hinzufügen und entfernen können

Funktionen

Funktionen sind wiederverwendbare Code-Blöcke, welche eine spezifische Aufgabe erfüllen.

  • Sie verhindern Code-Wiederholung
  • Erhöhen Lesbarkeit und Struktur
  • Erlauben komplexe Logik zu kapseln

Wie ist eine Funktion aufgebaut?

                    
                        function greet(name) {
                            console.log("Hallo " + name);
                        }

                        greet("Anna");

Return-Werte

Eine Funktion kann einen Wert zurückgeben, der weiterverwendet werden kann.

                    
                        function add(a, b) {
                            return a + b;
                        }

                        const result = add(5, 7);
                        console.log(result); // -> 12

Funktionstypen

  • Normale Funktionen (Function Declaration)
  • Anonyme Funktionen (Function Expression)
  • Arrow Functions
  • IIFE (Selbstausführende Funktion)

Normale Funktion

                    
                        function sayHi() {
                            console.log("Hi!");
                        }

                        sayHi();

Anonyme Funktion

Wird häufig Variablen oder EventListeners zugeordnet.

                    
                        const multiply = function(a, b) {
                            return a * b;
                        };

                        console.log(multiply(3, 4)); // -> 12

Arrow Function

Kürzere Schreibweise, kein eigenes this.

                    
                        const subtract = (a, b) => {
                            return a - b;
                        };

                        console.log(subtract(10, 4)); // -> 6

IIFE – Immediately Invoked Function Expression

Selbstausführende Funktion, nützlich zur Kapselung.

                    
                        (function() {
                            console.log("Ich führe mich selbst aus!");
                        })();

EventListeners

EventListeners reagieren auf Benutzeraktionen.

  • Klicks
  • Tastatureingaben
  • Mausbewegungen
  • Seiten-Interaktionen (z.B. Scroll)

Meist genutzte EventListener

click Reagiert auf Mausklicks
input Reagiert bei Benutzereingaben
submit Reagiert bei Formular-Übermittlung
mouseover Reagiert, wenn die Maus über ein Element fährt

EventListener definieren

                    
                        <button id="btn">Klick mich</button>
                    
                
                    
                        const btn = document.getElementById("btn");

                        btn.addEventListener("click", function() {
                            console.log("Button wurde geklickt!");
                        });

EventListener mit Arrow Function

                    
                        btn.addEventListener("click", () => {
                            console.log("Arrow Function Event!");
                        });

EventListener entfernen

Ein Event kann nur entfernt werden, wenn die Funktion benannt ist.

                    
                        function onClick() {
                            console.log("Click!");
                        }

                        btn.addEventListener("click", onClick);
                        btn.removeEventListener("click", onClick);

TL;DR

  • Funktionen kapseln wiederverwendbaren Code
  • Return-Werte geben Ergebnisse zurück
  • Funktionstypen: normal, anonym, Arrow, IIFE
  • EventListener reagieren auf User-Interaktionen
  • Events können hinzugefügt und entfernt werden
